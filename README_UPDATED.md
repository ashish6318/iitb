## ğŸš€ Quick Start

### Prerequisites
- Node.js 16+ 
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd frontend-battle-website
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

   ```bash
   npm run build
   ```

   ```bash

   ```
